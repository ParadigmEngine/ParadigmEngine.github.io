  • namespace core the engine core, here all graphics resources and render operations are described.

      • class clear_value typesafe alternative to vk::ClearValue
      • class framebuffer_t final container class that describes the data to create a set of rendertargets.

          • class description final describes how the rendertarget should be loaded, stored, it's sample count, and what transition layouts it should go through.
      • class geometry_t describes a stream of data that will be uploaded to the GPU as geometry data.
        • struct constants contains globally pre-defined keys for common streams that most engines have.

      • class computepass describes a compute stage in the render_graph
      • class drawgroup a collection of draw instructions to be recorded and sent to the GPU.

      • class buffer_t maps a memory region and interfaces with the driver for read/writes
      • class context encapsulated a graphics context.
      • class drawpass a pass describes a start-to-finish sequence of commands to render into a set of rendertargets.
      • class framebuffer_t final describes a set of images to use as rendertargets
        • struct binding describes a single binding point (can be many) in a framebuffer.
      • class geometry_t describes the driver visible concept of geometry.
      • class material_t final class that creates a bindable collection of resources that can be used in conjuction with a surface to render.

      • class sampler_t final sampler object for texture filtering and lookups
      • class shader creates a shader object from a SPIR-V module
        • struct specialization contains the specialization info that might be used by the shader.
      • class swapchain describes a framebuffer that is specially handled and created by the driver
      • class texture_t a texture resource used for rendering, either as target, or as input resource.
    • namespace meta contains all extensions to the ::meta namespace.
      • class shader final contains extensions for meta data when loading shader files.
      • class texture_t final Custom meta data that describes a texture. i.e. width, height, format, etc..
    • namespace os specific OS wrappers and resources.
