\doxysubsection{Detailed Description}
swpachains can be considered special \mbox{\hyperlink{classcore_1_1ivk_1_1framebuffer__t}{core\+::ivk\+::framebuffer\+\_\+t}}\textquotesingle{}s, with some special pecularities. For example, unlike framebuffer\textquotesingle{}s, you have to request the next image from the driver for a swapchain, and there are many more restrictions on both format and size imposed on swapchains.
