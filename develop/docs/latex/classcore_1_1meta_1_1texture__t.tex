\doxysubsection{Detailed Description}
does not contain any actual texture data, just describes it. This aids the engine in setting up the correct resources for rendering the associated texture file. \begin{DoxyWarning}{Warning}
The description is assumed to be correct, otherwise undefined behaviour happens. 
\end{DoxyWarning}
\begin{DoxyNote}{Note}
although this class allows runtime editing, it\textquotesingle{}s unlikely you\textquotesingle{}d need it unless you generate textures. and wish to serialize them to disk. 
\end{DoxyNote}
