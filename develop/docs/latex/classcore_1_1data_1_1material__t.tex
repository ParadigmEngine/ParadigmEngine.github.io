\doxysubsection{Detailed Description}
Material data describes a collection of textures, buffers, shaders, override parameters for these shaders, blend operations (such as opaque or transparent), render order offsets, and generic options of how this material {\itshape should} be rendered. It also contains facilities to set default values on buffers if needed.
