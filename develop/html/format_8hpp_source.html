9 /*
10  string based version:
11  { optional header }
12  MAJOR.MINOR.PATCH
13  { body }
14  value_t: [FLOAT]2.5[/FLOAT]
15  value_t (literal): [FLOAT]'''2.5'''[/FLOAT]
16  value_t: [SHADER]AF7D9C55-17C0-431A-BB4D-98A02F0FFB80[/SHADER]
17  value_range_t: [FLOATS]{1.4, 3.9}[/FLOATS] or [FLOATS]{'''1.4''', 3.9}[/FLOATS]
18  value_range_t (literal): [FLOATS]{'''1.4''', 3.9}[/FLOATS]
19  reference_t: [SHADER_NAME]&COLLECTION::CHILD_STRING[/SHADER_NAME]
20  reference_range_t: [REFERENCENODES]&{FLOAT, SHADER_NAME}[/REFERENCENODES]
21  collection_t: [COLLECTION][CHILD_STRING]I'm a child![/CHILD_STRING][/COLLECTION]
22 
23  compact based verion:
24  HEADER:
25  PCMPDATA ...
26 
27  Table of contents:
28  node name index, node content index, node type..
29 
30  content:
31  name
32  content
33  binary based version:
34  HEADER:
35  PBINDATA ...
36  Table of contents:
37  node name index, node content index, node type..
38 
39  content:
40  name
41  content
42 
43  internal data representation:
44  value_t: psl::string8::view -> view into the container where the content is.
45  value_range_t: vector<string_view> -> view into the container for each content element.
46  reference_t: nodes_t index of the node in the container vector
47  reference_range_t: vector<nodes_t> index range of all nodes it is referencing in the container vector
48  collection_t: pair<nodes_t, nodes_t> start index and size of the nodes in the parent (works like a range<T> )
49 */
