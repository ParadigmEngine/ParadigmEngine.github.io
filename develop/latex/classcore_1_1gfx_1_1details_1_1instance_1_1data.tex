\doxysubsection{Detailed Description}
Manages instance data related to materials-\/geometry, both the geometry specific data (such as instance position, etc...), as well as material-\/wide instance data (shared with all drawcalls using this specific instance set). The latter could be visualised as all pieces of foliage sharing the same \char`\"{}wind intensity\char`\"{} value. This is not to be confused as \char`\"{}global data\char`\"{}, such as all pieces of geometry knowing about \char`\"{}fog\char`\"{}, it is still duplicated over every instance of a bundle.
