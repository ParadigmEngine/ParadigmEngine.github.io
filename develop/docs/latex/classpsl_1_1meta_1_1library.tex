\doxysubsection{Detailed Description}
a \mbox{\hyperlink{classpsl_1_1meta_1_1library}{meta\+::library}} is a collection of \mbox{\hyperlink{classpsl_1_1meta_1_1file}{meta\+::file}} instances. The library will do the basic bookkeeping, allowing you to search and retrieve \mbox{\hyperlink{classpsl_1_1meta_1_1file}{meta\+::file}} instances based on their \mbox{\hyperlink{structpsl_1_1_u_i_d}{psl\+::\+U\+ID}}, or tags and narrow it down depending on the \mbox{\hyperlink{classpsl_1_1meta_1_1file}{meta\+::file}} type as well.

There are also facilities to have \mbox{\hyperlink{structpsl_1_1_u_i_d}{psl\+::\+U\+ID}}\textquotesingle{}s reference other \mbox{\hyperlink{structpsl_1_1_u_i_d}{psl\+::\+U\+ID}}\textquotesingle{}s, and do bi-\/directional lookups for these relations. As well as the ability to load the persistent companion file that the \mbox{\hyperlink{classpsl_1_1meta_1_1file}{meta\+::file}} describes (in case the \mbox{\hyperlink{structpsl_1_1_u_i_d}{psl\+::\+U\+ID}} satisfies \mbox{\hyperlink{classpsl_1_1meta_1_1library_a30001692a65acadf17d3b1315566aebd}{is\+\_\+physical\+\_\+file()}} ), and to have that companion file cached for faster reloads.
